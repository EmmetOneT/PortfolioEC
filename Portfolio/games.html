Drone Annihilation: Blacksite

My first person shooter made in Unreal Engine entirely using Blueprints. The game can be played with mouse and keyboard but also has full gamepad support. There is a graphics settings menu with all graphics settings expected from a modern game from texture quality to anti aliasing. There is an optimal settings button which gets the users hardware and sets the graphics accordingly. The player can save these settings. The game includes a timer in which the player must destroy all drones on the map to win. The game has been set up as a more oldschool shooter. There are ammunition, health and time pickups. I you get too close to a drone, they will navigate towards you before exploding. If a drone is shot but not destroyed, they will aggro and navigate towards the player.
